## FE News

### 1.[A big week for jQuery](https://javascriptweekly.com/issues/769)
> 相关地址：https://javascriptweekly.com/issues/769

jQuery 4.0 正式发布！时隔 20 年，这个广受欢迎的 JavaScript 库迎来了重大更新。新版本迁移至 ES 模块，与 Vite、webpack 等现代构建工具完美兼容。同时，jQuery 4.0 移除了对 IE 10 及更早版本的支持，专注于现代浏览器的优化。

这个发布标志着 jQuery 向现代化迈进的重要一步，使其能够更好地融入当今的前端开发生态。

### 2.[Can we ever fix the web dependency mess?](https://javascriptweekly.com/issues/768)
> 相关地址：https://javascriptweekly.com/issues/768

Web 依赖管理问题探讨 — Lea Verou 深入分析了 JavaScript 依赖管理的复杂性。作为 Web 标准领域的资深专家，Lea 从专业角度探讨了为什么 JavaScript 的依赖管理和引入会变得如此复杂，并提出了可能的解决方案。

这篇文章是每个 JavaScript 开发者必读的内容，深入剖析了我们日常开发中遇到的依赖管理痛点。

### 3.[A new JavaScript engine from Fabrice Bellard](https://javascriptweekly.com/issues/767)
> 相关地址：https://javascriptweekly.com/issues/767

著名开发者 Fabrice Bellard 发布了新的 JavaScript 引擎！作为 QEMU、FFmpeg 等知名项目的创造者，Fabrice Bellard 再次带来了令人期待的技术创新。

新引擎专注于性能和特性支持，为 JavaScript 生态系统注入了新的活力。值得关注的是其在特定场景下的性能表现和独特优化策略。

### 4.[The JavaScript Bundler Grand Prix](https://javascriptweekly.com/issues/766)
> 相关地址：https://javascriptweekly.com/issues/766

JavaScript 打包工具大比拼 — 2025 年度打包工具性能对比测试结果出炉！本期特别汇总涵盖了多个主流打包工具在不同场景下的性能表现。

测试涵盖了构建速度、产物体积、Tree-shaking 效果等多个维度，为开发者选择合适的打包工具提供参考依据。
