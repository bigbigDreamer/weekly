# 周刊生成工具 - 快速开始

## 一键生成新一期周刊

### 方式一：在 Claude Code 中使用（推荐）

直接对 Claude 说：

```
请帮我生成新一期的周刊
```

Claude 会自动：
- ✅ 获取最新期数
- ✅ 从 JavaScript Weekly 获取最新 7-10 条新闻
- ✅ 使用 AI 将英文翻译成流利的中文
- ✅ 智能提取关键词
- ✅ 生成格式化的周刊文件

### 方式二：使用 Python 脚本（备用）

```bash
python3 scripts/generate_weekly.py
```

## 生成的文件位置

```
docs/{年}/{月}/{期数}/index.md
```

例如：`docs/2026/01/62/index.md`

## AI 翻译特色

### 英语专八水平的翻译质量

✅ **流利的中文学表达**
- 避免"翻译腔"，使用地道的中文表达
- 技术术语准确，保持专业性
- 补充上下文，便于理解

### 翻译示例对比

❌ **差的翻译**：
```
jQuery 4.0 发布 — 20 年从它的原始发布，这个流行的库到达 4.0
```

✅ **AI 优质翻译**：
```
jQuery 4.0 正式发布！时隔 20 年，这个广受欢迎的 JavaScript 库迎来了 4.0 版本。
新版本迁移至 ES 模块，与 Vite、webpack 等现代构建工具完美兼容。
```

### 技术术语处理

**保留英文**：React, Vue, Angular, jQuery, API, npm, webpack

**翻译为中文**：
- framework → 框架
- library → 库
- released → 发布
- performance → 性能
- security → 安全
- breaking changes → 破坏性变更

## 后续步骤

生成文件后，建议手动：

1. **添加封面图片**
   ```markdown
   ![](https://your-image-url.png)
   (封面图片来源于：xxx)
   ```

2. **检查翻译质量**
   - 确认中文表达流畅
   - 检查技术术语准确性
   - 补充重要细节

3. **添加特性列表**（如果适用）
   ```markdown
   主要变化：
   - 迁移至 ES 模块
   - 移除 IE 支持
   - 性能优化
   ```

4. **调整关键词**
   - 确保准确反映本期重点

## 常见问题

**Q：翻译质量如何保证？**
A：Claude 具备英语专八水平，结合深厚的技术背景，翻译质量远超机器翻译。

**Q：需要安装依赖吗？**
A：不需要，Claude Code 直接使用 AI 模型完成所有工作。

**Q：可以自定义翻译风格吗？**
A：可以，告诉 Claude 你的偏好即可（如"更口语化"、"更专业"等）。

**Q：支持哪些 Python 版本？**
A：如果使用 Python 脚本，需要 Python 3.6+；使用 Claude 则不需要。

**Q：生成的文件可以直接发布吗？**
A：建议添加封面图片并检查翻译后再发布。
