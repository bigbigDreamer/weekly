## 概述

这是一套专门为 ryanwang's weekly 设计的 AI 驱动周刊生成工具。与传统的脚本工具不同，本工具充分利用 Claude Code 的 AI 能力，提供**英语专八水平的翻译质量**和**智能内容创作**。

## 核心特性

✨ **AI 驱动**：由 Claude AI 直接完成 RSS 获取、翻译和内容生成
✨ **优质翻译**：英语专八水平 + 技术专家背景，翻译质量远超机器翻译
✨ **零依赖**：无需安装 Python 或任何外部依赖

✨ **一键生成**：简单对话即可完成全部工作

## 工作原理

### AI 处理流程

```
用户请求
  ↓
Claude 查找最新期数（Glob 工具）
  ↓
获取 RSS 内容（WebFetch 工具）
  ↓
AI 阅读并理解英文内容
  ↓
AI 翻译成流利中文（英语专八水平）
  ↓
AI 智能提取关键词
  ↓
生成格式化文件（Write 工具）
  ↓
提供预览和优化建议
```

### 技术栈

- ✅ **Claude Code AI 模型**：核心处理能力
- ✅ **WebFetch 工具**：获取 RSS 内容
- ✅ **Glob 工具**：查找文件
- ✅ **Write 工具**：创建文件
- ✅ **Bash 工具**：执行命令（创建目录等）
- ❌ **无需 Python**
- ❌ **无需外部翻译 API**
- ❌ **无需任何依赖**
